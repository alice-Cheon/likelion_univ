벤더 프리픽스(Vendor Prefix)란?

벤더 프리픽스(vendor prefix)란 주요 웹 브라우저 공급자가
새로운 실험적인 기능을 제공할 때 이전 버전의 웹 브라우저에
그 사실을 알려주기 위해 사용하는 접두사(prefix)를 의미합니다.
아직 css 권고안에 포함되지 않은 기능이나, 포함되어 있지만
아직 완벽하게 제정된 상태가 아닌 기능을 사용할 때 사용합니다.
 - 해당 속성이 적용되지 않았을 경우를 표현해야 할 코드를
   가장 먼저 작성해야 하며, 표준 문법 코드는 가장 마지막에 작성해야 합니다.

 사용법

* 브라우저별 프리픽스
